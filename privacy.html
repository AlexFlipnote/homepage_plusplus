Privacy Policy for Homepage++

Last updated: 21. November 2025

Homepage++ respects your privacy.
This extension does not collect, store, or transmit any personal data.
All data and settings are stored locally on your device and are never sent to any external server.

Permissions Used

  • storage: Used to save your custom backgrounds, preferences, and other local settings.
  • geolocation: Used only to show weather and location information when the weather feature is enabled. Your location data is never stored, it's used temporarily by your browser to fetch the weather from MET Norway and the location data from OpenStreetMap with your coordinates. If weather is turned off or you have enabled manual location, geolocation is not used.
  • topSites: Used to fetch your most visited sites for displaying bookmarks if you enable that feature. If disabled, this permission is not used.
  • favicon: Used to fetch the favicon of your most visited sites for displaying bookmarks. If disabled, this permission is not used.
  • search: Used to perform searches using the browser's default search engine.

Host Permissions

This extension requests the following host permissions:

Host Purpose
https://api.met.no/* To fetch weather data when the weather feature is enabled.
https://*.openstreetmap.org/* Fetch location data when weather is enabled and show map chunks when using the map feature.

Data Collection

This extension does not:

  • Collect, store, or transmit any personal or usage data.
  • Use analytics, tracking, or third-party scripts.
  • Connect to external servers, except when fetching weather data (via your browser's built-in API).
